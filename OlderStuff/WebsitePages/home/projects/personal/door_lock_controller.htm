Door Lock Control Software to communicate with door lock

My friends and I developed an android application for controlling a locking mechanism for a door. The application gains secure access to an Adafruit Feather HUZZAH with ESP8266 WiFI. The purpose of using this device is to connect to the door lock via wifi or anywhere as long as you have access to the internet.

My friend Christian Steinmetz is an electrical engineering and audio technology major at Clemson University. We split up the work based on my background in coding and his in hardware and coding for microcontrollers.

Originally, we started the project using the Lua programming language along with the nodeMCU as the language for the microcontroller, but we switched the code to MicroPython, because we were encountering problems that were common with Nodemcu's ssl library and also micro python seemed like a really cool and generally more well supported language.
